Project
Obstacle Avoid Moving Car

"The Obstacle Avoiding Moving Car" is a microcontroller-based robotics project developed during my college coursework. The system uses an ultrasonic sensor to continuously measure the distance between the car and nearby objects. When an obstacle is detected within a predefined range, the controller processes the sensor data and automatically alters the motor direction to avoid collision. This project helped me understand sensor integration, motor control, embedded programming, and real-time decision making. It demonstrates the fundamentals of autonomous systems and robotics, which are widely used in modern automation and self-driving technologies.Developed an autonomous obstacle-avoiding moving car as a college project using a microcontroller and ultrasonic sensor.
